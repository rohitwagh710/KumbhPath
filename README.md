# KumbhPath – Smart Mobility Platform for Kumbh Mela

KumbhPath is a smart mobility, transportation, accommodation and navigation platform designed to support millions of pilgrims visiting the Nashik Kumbh Mela.

The platform helps pilgrims easily navigate the city, access transport services, locate facilities, and stay connected with their family members in crowded environments.

---

## Key Features

• Real-time tracking of government and private transport  
• Smart navigation to ghats, temples and transport hubs  
• Family tracking to locate loved ones in crowded areas  
• Multilingual interface for pilgrims from different states  
• Emergency support for medical and security services  
• AI assistant for schedules, announcements and guidance  

---

## System Components

KumbhPath consists of three integrated systems:

1. Passenger Application  
2. Driver Application (Government + Private)  
3. Admin Dashboard for Authorities  

---

## Technology Stack

• Flutter – Mobile application development  
• Firebase – Authentication & real-time database  
• Google Maps API – Navigation and location services  
• Python / FastAPI – Backend processing  
• Razorpay – Secure payment integration  

---

## Architecture

The system architecture includes:

Passenger App → Backend Server → Database → Admin Dashboard → Driver App
